# Placeholder para vídeos de demonstração

## Como adicionar vídeos reais:

### Opção 1: Usar o Gerador HTML
1. Abra o arquivo `create-demo-videos.html` no navegador
2. Selecione um exercício
3. Clique em "Gerar Vídeo"
4. Faça download e renomeie para .mp4

### Opção 2: Baixar vídeos do YouTube
Use o youtube-dl ou similar para baixar os vídeos das URLs mapeadas:

```bash
# Exemplo para instalar youtube-dl
pip install youtube-dl

# Baixar vídeo específico
youtube-dl -f mp4 "https://www.youtube.com/watch?v=esQi683XR44" -o "bench-press.%(ext)s"
```

### Opção 3: Criar vídeos com ffmpeg
Se tiver ffmpeg instalado:

```bash
# Criar vídeo de placeholder colorido
ffmpeg -f lavfi -i color=c=green:size=640x360:duration=10 -c:v libx264 -t 10 -pix_fmt yuv420p bench-press.mp4

# Adicionar texto
ffmpeg -f lavfi -i color=c=black:size=640x360:duration=10 -vf "drawtext=fontfile=/System/Library/Fonts/Arial.ttf:text='Supino Reto':fontcolor=white:fontsize=24:x=(w-text_w)/2:y=(h-text_h)/2" -c:v libx264 -t 10 -pix_fmt yuv420p bench-press.mp4
```

### Opção 4: Usar vídeos de stock
Baixe vídeos gratuitos de exercícios de sites como:
- Pexels Videos
- Pixabay Videos  
- Unsplash Videos
